Logger Module

Initialize decorators package and expose public logging utilities.

jl.decorators.log_execution(func: T) → T[source]

Decorator that logs execution details of a function or method.

This includes entry, arguments, return value, execution time, and exceptions if any occur during execution.

Parameters:

func (T) – The function or method to be wrapped.

Returns:

The wrapped function with logging.

Return type:

T
